# 从 npm (opens new window) 下载的组件库或者第三方库

也就是您使用 pnpm add 添加的包，比如 vue-router (opens new window)，我们只需要将这个包提供的全局类声明文件导入到 tsconfig.json (opens new window) 的 types 配置项，然后重启编辑器即可，如下图导入前和导入后的效果对比

导入前，pure-table 无高亮且鼠标覆盖无类型提示

导入后，pure-table 高亮且鼠标覆盖有类型提示
